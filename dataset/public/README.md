# Challenge data (from the organizer)

Keep the released files here (same layout as the competition zip `dataset/public/`):

- `train.jsonl` — labeled training rows
- `test.jsonl` — test prompts and **visible** examples only
- `sample_submission.csv` — CSV shape example

Code loads these via `golf_ft.paths` (`TRAIN_JSONL`, `TEST_JSONL`, `SAMPLE_SUBMISSION_CSV`). Do not maintain duplicate copies at the repo root.
